# README

## Criar ambiente virtual do projeto

```
python3 -m venv .venv
```

## Ativar ambiente virtual do projeto

```
.venv/bin/activate
```

## Atualizar Pip

```
pip install --upgrade pip
pip install -U flake8
```

## Instalar pre-commit

```
pip install pre-commit
```

## Executar pre-commit

```
pre-commit
```
